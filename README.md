# 🌐 Personal Portfolio Website

Modern **portfolio website** designed to showcase my work as a
**software developer**. The site highlights my background, skills, projects,
and services, using a clean UI and smooth animations for a great user
experience. Optimised for desktop, tablet, and mobile devices.

## 🚀 Live Website

Go to website: [joherrer.com](https://joherrer.com)

## 📋 Overview

This portfolio includes:

- **Homepage** – Intro hero section, animated typing text, social links,
  and visuals
- **About** – Background, education timeline, skills, fun facts
- **Resume** – Education, software & tools, technical and professional skills
- **Services** – Web development, desktop apps, AI solutions, CI/CD, cloud,
  business tools
- **Projects** – Filterable portfolio built with Jekyll collections
- **Contact** – Contact details + working contact form

Powered by **Jekyll**, fully compatible with **GitHub Pages**.

## ✨ Features

- ⚡ Fast & lightweight static site
- 🎨 Modern Bootstrap 5 UI
- 📱 Fully responsive design
- 🧩 Dynamic portfolio with filters
- 🎥 Smooth animations
- 📩 Functional contact form with anti-spam
- 📦 Clean and extensible codebase

## 🛠️ Tech Stack

### **Frontend**

- HTML, CSS, JavaScript
- Bootstrap 5
- AOS (Animate On Scroll)

### **Backend**

- Jekyll
- Liquid Templates
- YAML configuration & collections
