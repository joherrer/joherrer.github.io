# 🌐 Personal Portfolio Website

A modern, responsive portfolio website built to showcase my work as a **Software Developer**, including my background, skills, projects, and the services I offer. This website serves as my digital presence, combining clean UI, smooth animations, and structured content to reflect professionalism and creativity.

## 🚀 Live Website

Go to website: [https://joherrer.github.io/](https://joherrer.github.io/)

## 📋 Overview

This portfolio includes:

- **Homepage** – Intro hero section, animated typing text, social links, and visuals
- **About** – Background, education timeline, skills, fun facts
- **Resume** – Education, software & tools, technical and professional skills
- **Services** – Web development, desktop apps, AI solutions, CI/CD, cloud, business tools
- **Projects** – Filterable portfolio built with Jekyll collections
- **Contact** – Contact details + working Formspree form

Powered by **Jekyll**, fully compatible with **GitHub Pages**.

## ✨ Features

- ⚡ Fast & lightweight static site
- 🎨 Modern Bootstrap 5 UI
- 📱 Fully responsive design
- 🧩 Dynamic portfolio with filters 
- 🎥 Smooth animations
- 📩 Functional contact form with anti-spam
- 📦 Clean and extensible codebase

## 🛠️ Tech Stack

### **Frontend**
- HTML5, CSS3, JavaScript
- Bootstrap 5  
- AOS (Animate On Scroll)

### **Backend / Engine**
- Jekyll
- Liquid Templates
- YAML configuration & collections

### **Tools**
- Git & GitHub Pages  
- Visual Studio Code
